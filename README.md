# PhishIntel AI: Hybrid Phishing Detection System

A high-performance web application that uses **Machine Learning (Logistic Regression)** and **WHOIS Metadata** to detect phishing URLs in real-time.

## 🚀 Features
- **Hybrid Detection:** Combines AI pattern recognition with live Domain Age verification.
- **Dynamic Risk Scoring:** Uses probability-based confidence intervals.
- **Incident Logging:** Categorized history for "Verified Safe" and "Threat History."
- **Modern UI:** Responsive dark-mode dashboard.

## 🛠️ Installation
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Run the trainer: `python train_ai.py`
4. Start the app: `python app.py`
