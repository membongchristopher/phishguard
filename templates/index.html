About the System

How the AI Powered Phishing Detection System works

Hybrid AI Pipeline

Combines TF-IDF text analysis of URL tokens with 15 hand-crafted numerical features into a 50,015-dimension feature matrix fed into machine learning models.

Random Forest

Best-performing model with 98.1% test accuracy and AUC-ROC of 0.994, trained on 549,346 labelled URLs from PhishTank and legitimate URL datasets.

Brand Spoofing Detection

Detects URLs where a known brand (PayPal, Google, Apple) appears in the subdomain but not the actual registered domain — a classic phishing technique.

Suspicious TLD Detection

Flags domains using free or abused Top-Level Domains like .tk, .xyz, .ml and .ga which are disproportionately used in phishing campaigns.

IP Address Detection

Identifies URLs using a raw IP address as the domain (e.g. http://192.168.1.1/login). Legitimate websites always use registered domain names.

Flask REST API

Python Flask backend. The /analyze endpoint runs tokenization, TF-IDF, feature extraction, scaling, and model inference — returning a full JSON risk report.
